IoT Cat Laser Toy Project

Project Overview

The IoT Cat Laser Toy is an interactive pet toy with 360-degree stepper motor rotation and servo-controlled laser movement, controllable via a mobile app. This smart toy will allow pet owners to entertain their cats remotely and implement adaptive play algorithms based on their cat's play patterns.

This project is in progress right now.

Stay tuned for updates as soon as spring 2025!
